# SnapIt📸

- 여러 포토 부스에서 촬영한 사진을 모아 볼 수 있는 모바일 어플리케이션
- 포토 부스에서 촬영한 사진의 QR 코드를 통해 바로 어플리케이션 사진과 영상을 다운로드할 수 있음
- 기본 카메라에서 QR 코드 촬영시 어플리케이션에 자동 등록

## :star: 구현 완료 목록 :star:
- **Backend**
1. 사용자 로그인, 로그아웃, 회원가입
2. 사용자 프로필 변경
3. 사용자의 이미지, 동영상 불러오기

- **Frontend**
1. 메인 화면 구현
2. 튜토리얼 화면 구현
3. 로그인, 회원가입 화면 구현
4. 사용자 프로필, 프로필 수정 화면 구현
